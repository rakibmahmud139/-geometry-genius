1. What is DOM and what is the purpose of DOM?

Ans : Dom is a programming interface for web documents. It represents the page so that programs can change the document structure, style and content

2. How will you select HTML elements using DOM? Name the DOM methods?

Ans : There are 5 ways in which we can select elements in a DOM using selectors .
1. getElementsByTagName()
2. getElementsByClass()
3. getElementById()
4. querySelector()
5. querySelectorAll()

3. What is event bubble?

Ans : Event Bubbling is a concept in the DOM . It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

4. What is a callback function and why will you use it?

Ans : A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action. Callback functions are needed because many Javascript actions are asynchronous.
